# ToDo アプリ計画書 v1.0

更新日: 2025/10/28

## 概要

軽量な ToDo アプリ。シンプルな UI で、タスクの編集・追加・完了・検索・タグ管理を高速に行える。
将来的には、通知や共有機能を実現したいところ。

主要価値
・起動や操作が高速
・あとから拡張しやすい

## ターゲット

個人ユーザー(学生/個人開発者)
課題：軽量・素早い入力
解決方針：ショートカット・最近タグ提案によって入力の摩擦を最小化

## 開発プラン

### メイン機能

・タスクの追加・編集・完了・削除
・検索
・タグ管理
・永続化
・UI(ダーク/ライト)
・実機テスト(iPhone)

### サブ機能

・通知(毎日/期限前)
・並び変え(ドラッグアンドドロップ)
・エクスポート/インポート(JSON)

## 名称アイデア

・LiteDo
